How it works

01

Install once

One command installs the binary and registers it with Claude Code, Kiro, and opencode — works in every project from then on. Run pasteai doctor any time to diagnose issues.

terminal 
$ curl -sSL https://github.com/
  pasteai/pasteai/install.sh | sh

or in Claude Code:

/setup

already installed?

pasteai setup
02

Just ask

Tell your AI to publish something. It handles the rest.

you → Claude 
Summarise today's planning session
and publish it as a shareable doc.

↓

Done — here's your link:
localhost:8080/d/a3b4c5
03

Share the link

Anyone with the link gets a clean, readable page — no login required.

shareable URL 
localhost:8080/d/
planning-session-a3b4c5

✓ Rendered markdown
✓ Syntax highlighting
✓ Table of contents
